# python-cli-elasticsearch-multi-node-without-ssl-client-simple

## Description
Reads a multi node cluster for data in `dog-demo` document.

## Tech stack
- python
- elasticsearch
- kibana

## Docker stack
- python
- elasticsearch
- kibana

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
